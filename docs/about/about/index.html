About

My name is Michael, and my professional background is in .NET software engineering (C#, MVC, MSSQL, etc), though this is within the context of an ecosystem of other technologies related to deployment, version control, distribution and testing. Aside from that, I do have some expertise with the Linux operating system (mostly variants of Debian and openSUSE), which I have been using since ~2005.

After several years working for the public sector, where I provided technical support for secure communications networks, I decided to take this in a more academic direction, studying computer security at the University of South Wales, and managing to earn a Masters’ degree in 2014.

As a software engineer, my current position involves addressing legacy software, technical debt, technical documentation, systems security and advising on engineering best practices. Essentially I’m using the latest technologies to make existing systems more robust, secure and maintainable.

Outside of work, I’m a martial artist (pandemic-related hiatus aside), and a proud owner of an MX-5.
